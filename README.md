# Software Requirements

To execute the attached code the following python packages must be installed. The software was tested with the designated version numbers. Datasets used in the analysis can be found at https://ega-archive.org/studies/EGAS50000001644. Installation of these packages should take less than one hour. All packages can be installed with "pip install".
All attached code can be run in the provided jupyter notebook using standard practices, assuming the file paths of the downloaded data is adjusted for your computer and requires no special installation to run. No non-standard hardware is required. Most jupyter cells will run in under 5 minutes; however, the isolation of telomeric sequences from NGS may take up to several hours per cell.
